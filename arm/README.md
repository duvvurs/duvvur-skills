# ARM Templates — Classic Azure IaC

Legacy ARM (JSON) templates for FinOps governance resources.

> **Note:** New deployments should use [Bicep](../bicep/). ARM templates here are for environments where Bicep adoption is in progress.

## Templates

| File | Purpose |
|------|---------|
| [`templates/`](templates/) | Budget creation, Cost Management export, Policy assignment |

See [Bicep policies](../bicep/policies/) for the modern equivalent of these patterns.
