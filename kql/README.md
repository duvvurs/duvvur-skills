# KQL — Azure Resource Graph Queries

Production Kusto Query Language patterns for Azure cost visibility, governance, and optimization.

## Categories

| Area | Queries | Purpose |
|------|---------|---------|
| [**Cost Analysis**](cost-analysis/) | Daily burn rate, cost by tag, anomalies, top spenders | Where is the money going? |
| [**Governance & Compliance**](governance-compliance/) | Tag audit, policy compliance, RBAC drift, stale resources | Is governance working? |
| [**Optimization**](optimization/) | Idle VMs, unattached disks, old snapshots, right-sizing | Where can we save? |
| [**Security**](security/) | Unencrypted expensive resources, open NSGs, no backup | Security-informed cost review |

Every query includes the business question it answers and the action it drives.
