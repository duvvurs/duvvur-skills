# Power BI — Azure Cost Analytics

Semantic models, DAX measures, and dashboard patterns for Azure cost reporting.

## Components

| Area | Description |
|------|-------------|
| [**DAX Measures**](dax-measures/) | 20+ production measures: MoM change, projected annual spend, RI coverage, budget variance |
| [**Dataset Schema**](dataset-schema/) | Star schema design with fact/dimension tables for Cost Management API data |

Built and refined across Power BI dashboards for European insurance and UK utilities clients.
