# PowerShell — Azure Automation

Production scripts for cost governance enforcement and reporting.

## Categories

| Area | Scripts | Purpose |
|------|---------|---------|
| [**Cost Management**](cost-management/) | Cost data export, budget alerting | Data pipeline for Power BI + proactive alerts |
| [**Governance**](governance/) | Tag compliance audit + enforcement | Drive tagging from 0% to 90%+ |
| [**Automation**](automation/) | Weekly FinOps report, scheduled runbooks | Automated governance cadence |

All scripts use `Az.*` modules and work with Azure Automation managed identity.
